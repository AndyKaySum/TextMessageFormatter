Getting started

  1. Include the CSS file "TextMessageFormatter.css" in any HTML files that may display TMF elements.
  2. Include the javascript file "TextMessageFormatter.js"

How to use Text Message Formatter

  1. Create an instance of TextMessageFormatter:

    const tmf = new TextMessageFormatter();

  2. The TextMessageFormatter instance (tmf in our case), has an attribute called messageContainer. This is the DOM element that will contain all of our messages. Append it to whatever DOM element you wish to use to house your messaging interface.

    For this guide, we have a Div in our DOM that we would like to use for this.

    divInOurDom.appendChild(tmf.messageContainer);

  3. To add messages to our message container, we need to first create an instance of a TMFMessage and include the contents of the message, the orientation (which side of the message container, our message should appear on), the timestamp of the message (if enabled), and the name of the sender (if enabled).

    const message = new TMFMessage("Hello world!", "left", "Now", "John Doe");

    Then, we need to call addMessage() from our TextMessageFormatter instance.

    tmf.addMessage(message);

Customizing the appearance of Text Message Formatter

  • To customize the appearance of Text Message Formatter elements, simply override the CSS style of classes (or change the values of variables) from the "TextMessageFormatter.css" file

  • This can be done by including the link to your own custom "override" CSS file underneath the link to "TextMessageFormatter.css"

    Alternatively, inline CSS or using a style tag can be used in the HTML file that displays Text Message Formatter elements.

Now you have everything you need to start working with TextMessageFormatter!
